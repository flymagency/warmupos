Comment configurer votre Geelark
1
Créer un compte Geelark
Rendez-vous sur geelark.com et créez votre compte. Choisissez un plan selon le nombre de profils souhaités.
2
Ajouter vos profils Instagram/Threads
Dans Geelark → Profiles → New Profile. Installez Instagram, connectez-vous à votre compte et notez l'ID numérique de chaque profil.
3
Importer le Custom Task WarmupOS
Téléchargez le fichier JSON ci-dessous et importez-le dans Geelark → Automation → Custom tasks → Import.
4
Configurer la tâche récurrente
Cliquez sur les 3 points du flux → Create recurring task → Interval → 120 minutes → sélectionnez vos profils → Publish.
5
Ajouter vos profils dans le dashboard
Revenez sur l'onglet "Mes comptes" → cliquez sur "+ Ajouter" → entrez le nom et l'ID Geelark de chaque profil.
